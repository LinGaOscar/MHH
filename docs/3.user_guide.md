# 3. 使用者操作手冊 (User Guide)

本文件說明 MHH 電文歷史平台的日常操作方式，適用對象為**分行經辦、分行主管、參數經辦、參數主管**。

---

## 1. 系統登入

- 本系統透過 **EIP SSO** 進行身分驗證，**不需要額外輸入密碼**。
- 登入後，系統會自動檢查您的 **SWALLOW 權限** 與 **HR 在職狀態**。
  - 若狀態異常（如已離職），系統將拒絕登入並顯示提示訊息。
- 若您擁有**跨國別權限**，請由右上角的國別切換選單切換。
  > ⚠️ 切換國別時，系統將重置登入狀態並變更語系設定。

---

## 2. Dashboard 儀表板

登入後進入儀表板，各區域說明如下：

| 區塊 | 說明 | 適用角色 |
|:--|:--|:--|
| **代辦事項** | 顯示您目前需要處理的電文或申請審核任務 | 所有角色 |
| **待放行提示** | 顯示待您審核的申請案（批次下載、自訂角色） | 主管角色 |
| **下載進度** | 顯示您已提交之預約合併下載的執行狀態與下載連結 | 經辦角色 |

---

## 3. 電文查詢與下載

### 3.1 查詢電文

1. 進入「電文查詢」頁面。
2. 輸入查詢條件（可組合）：
   - Sender BIC / Receiver BIC
   - 日期區間
   - 電文類型（MX / MT）
3. 點選「查詢」，結果以分頁清單顯示。

### 3.2 單筆下載 / 預覽

- 在查詢結果列表中，點選「PDF 預覽」可在瀏覽器內直接預覽文件。
- 點選「下載」可直接下載單筆 PDF，此動作會自動記錄至稽核日誌。

### 3.3 多筆預約下載

> 多筆下載需經主管審核放行，請依以下流程操作：

1. 在查詢結果中**勾選多筆**電文。
2. 點選「預約下載」按鈕，確認申請內容後送出。
3. 系統建立審核任務，狀態變為「**待放行**」。
4. 主管於 Dashboard 審核並點選「**放行**」。
5. 系統背景自動執行 PDF 合併作業。
6. 合併完成後，您可於 Dashboard「下載進度」區域點選下載最終合併檔。

---

## 4. 審核流程 (Maker-Checker)

MHH 採用 **Maker-Checker 雙人複核機制**，確保關鍵操作經過審核：

### 4.1 多筆電文下載審核

| 角色 | 操作 |
|:--|:--|
| **分行經辦（Maker）** | 發起多筆下載申請，狀態更新為「待放行」 |
| **分行主管（Checker）** | 於待辦事項中審核，確認後點選「放行」 |

### 4.2 參數維護審核

| 角色 | 操作 |
|:--|:--|
| **參數經辦（Maker）** | 發起「自訂角色申請」或「`USER_CUST` 維護申請」 |
| **參數主管（Checker）** | 於待辦事項中審核角色或自訂設定，確認後點選「放行」 |

> 所有放行與拒絕動作均會完整記錄至 `USER_LOGS` 稽核日誌。

---

## 5. 權限管理

- **查看權限**：進入「參數設定」頁面，可檢視您目前的角色與功能對應清單。
- **申請提升權限**：若需要提升至更高權限（如跨國別），可透過頁面發起申請，經主管審核後生效。
- **國別隔離原則**：跨國別的資料存取受到嚴格限制，系統自動依登入國別過濾可見資料範圍。
