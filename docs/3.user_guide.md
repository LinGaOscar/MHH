# 3. 使用者操作手冊 (User Guide)

本文件說明 MHH 電文歷史平台之日常操作流程，適用對象包括分行經辦、分行主管、參數經辦及參數主管。

---

## 1. 系統登入

- 系統採用 **EIP SSO** 身分驗證機制，無需額外密碼。
- 登入時，系統自動檢核 **SWALLOW 權限** 與 **HR 在職狀態**。
  - 狀態異常（如离職）者，系統將拒絕登入並顯示提示。
- 具備**跨國別權限**之使用者，可透過右上角選單切換國別。
  > ⚠️ 切換國別將重置登入狀態並變更語系設定。

---

## 2. Dashboard 儀表板

儀表板各區域功能說明如下：

| 區塊 | 說明 | 適用角色 |
|:--|:--|:--|
| **待辦事項** | 顯示待處理之電文或申請審核任務 | 所有角色 |
| **待放行提示** | 顯示待審核之申請案（如批次下載、自訂角色等） | 主管角色 |
| **下載進度** | 顯示已提交預約合併下載之執行狀態與下載連結 | 經辦角色 |

---

## 3. 電文查詢與下載

### 3.1 查詢電文

1. 進入「電文查詢」頁面。
2. 設定查詢條件（可組合）：
   - Sender BIC / Receiver BIC
   - 日期範圍
   - 電文類型（MX / MT）
3. 執行「查詢」，結果將採分頁清單顯示。

### 3.2 單筆下載與預覽

- 點選「PDF 預覽」可於瀏覽器直接檢視文件。
- 點選「下載」可取得單筆 PDF，系統將自動記錄於稽核日誌。

### 3.3 多筆預約下載

> 多筆下載需經主管審核放行，流程如下：

1. 於查詢結果中勾選目標電文。
2. 點選「預約下載」，確認申請內容後送出。
3. 系統產生審核任務，狀態設為「**待放行**」。
4. 主管於 Dashboard 進行「**放行**」操作。
5. 背景程序自動執行 PDF 合併。
6. 完成後可於 Dashboard「下載進度」區塊下載合併檔。

---

## 4. 審核流程 (Maker-Checker)

本平台採用 **Maker-Checker 雙人複核機制**，確保關鍵操作之合規性：

### 4.1 多筆電文下載審核

| 角色 | 操作 |
|:--|:--|
| **分行經辦 (Maker)** | 發起多筆下載申請，狀態轉為「待放行」 |
| **分行主管 (Checker)** | 於待辦事項審核並執行「放行」 |

### 4.2 參數維護審核

| 角色 | 操作 |
|:--|:--|
| **參數經辦 (Maker)** | 發起「自訂角色申請」或「`USER_CUST` 維護申請」 |
| **參數主管 (Checker)** | 於待辦事項審核設定內容，確認後執行「放行」 |

> 所有放行與拒絕紀錄均留存於 `USER_LOGS` 稽核日誌。

---

## 5. 權限管理

- **檢視權限**：於「參數設定」頁面檢視當前角色與授權功能清單。
- **權限變更**：若需提升權限（如跨國別存取），應發起申請並經主管審核生效。
- **國別隔離**：系統依登入國別自動過濾資料範圍，確保資料存取之安全隔離。
